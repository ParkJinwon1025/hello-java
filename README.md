## 1. Git clone

1. `git clone` 명령어 실행 실행
```bash
git clone https://github.com/ParkJinwon1025/hello-java.git
```

2. 디렉토리 이동
```bash
cd hello-java
```

3. `VS Code` 실행
``` bash
code .
```

1. `Terminal`에서 서버 명령어 실행
